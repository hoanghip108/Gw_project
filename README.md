## Introduction

## Technologies used in this project

Language: Javascript
Framework: ExpressJs
Database: MySQL (Sequelize)

## Project Preparing

```
Manually create database with name is in your environment file

```

Create .env file in root directory below format:

```
APP_NAME =
HTTP_HOST =
HTTP_PORT =
DB_HOST =
DB_PORT =
DB_DIALECT =
DB_USERNAME =
DB_PASSWORD =
DB_DATABASE =
JWT_SECRET =
PAGE_SIZE =
MAIL =
MAIL_PASSWORD =

```

## Setting up project

```shell
>npm install
```

## Starting project

```shell
> npm start
```

## Testing

```shell
> npm test
```
